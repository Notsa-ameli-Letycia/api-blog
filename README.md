# API Blog - Node.js & MySQL

## 🔥 Description
Cette API permet de gérer un blog simple.  
Elle permet de **créer**, **lire**, **modifier**, **supprimer** et **rechercher** des articles.  
Chaque article contient : titre, contenu, auteur, date de création, catégorie et tags.

Cette API a été développée dans le cadre du TAF1 de l'UE INF222 en utilisant Node.js et MySQL.

---

## 🛠️ Technologies utilisées
- **Node.js** avec **Express**
- **MySQL** pour la base de données
- **Postman** pour tester les endpoints
- (Optionnel) **Swagger** pour la documentation des endpoints
